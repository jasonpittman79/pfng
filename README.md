# pfng
My Portfolio in Angular 14

Build hosted at https://www.jasonpittman.dev/

To build locally to dist/, use the following command:

`ng build --base-href ./`

The portfolio features the following:

- Angular 14
- TypeScript
- SASS/SCSS
- HTML5
- Material UI
- Chart.js
- Responsive Design

# In the works
